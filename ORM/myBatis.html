데이터 연결 # 
                #은 데이터 값에 문자열을 자동으로 추가한 상태로 대입된다.
# 
                $는 데이터 값이 그대로 대입된다.

데이터 치환 태그를 삭제하고 싶은 경우 
                ● REGEXP_REPLACE(칼럼명, 정규식, 교체 내용) 함수를 사용한다.

                예시)
                    /* html 태그를 빈 문자열로 교체한다. */
                    REGEXP_REPLACE(title, '<[^>]*>|\&([^;])*;', '') LIKE '%${search}%'
특수문자를 사용하고 싶은 경우 
                ● <![CDATA[ 내용 ]]>을 사용한다.

                예시)
